# HTR pour l'écriture de Jean Calvin

### Modèles HTR pour l'écriture de Jean Calvin

Les modèles suivants ont été entrainés sur la plateforme eScriptorium avec Kraken et à l'aide d'un modèle entraîné sur l'écriture de Jean Calvin (13 pages manuscrites jusqu’à présent) à partir du model [HTR-United - Manu McFrench V1 (Manuscripts of Modern and Contemporaneous French)](https://zenodo.org/records/6657809). Vous pouvez consulter le protocole de transcription [ici](https://lettres-calvin.netlify.app/fr/recherche#protocole_transcription).

- [Calvin_manuscrits_tuned04](https://github.com/pippaJeanne/HTR-Jean-Calvin/blob/main/calvin_manuscrits_tuned04.mlmodel) - 93.9% précision
- [Calvin_manuscrits_tuned02](https://github.com/pippaJeanne/HTR-Jean-Calvin/blob/main/calvin_manuscrits_tuned02.mlmodel) - 92.7% précision
- [Calvin_manuscrits_tuned01](https://github.com/pippaJeanne/HTR-Jean-Calvin/blob/main/calvin_manuscrits_tuned01.mlmodel) - 89.4% précision
- [Calvin_manuscrits_tuned03](https://github.com/pippaJeanne/HTR-Jean-Calvin/blob/main/calvin_manuscrits_tuned03.mlmodel) - 84.7% précision

Voici le projet d'édition de la correspondance française de Jean Calvin pour lequel les modèles sont utilisés => [Lettres de Calvin](https://lettres-calvin.netlify.app)
